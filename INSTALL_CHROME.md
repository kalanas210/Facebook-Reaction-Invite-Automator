### "Manifest file is missing or unreadable"
- **Solution**: Make sure you selected the folder containing `manifest.json`
- Don't select a parent folder or a subfolder

### "This extension may have been corrupted"
- **Solution**: 
  - Click the refresh icon on the extension card
  - Or remove and reload the extension

### Extension icon doesn't appear
- **Solution**: 
  - Refresh the extensions page
  - Make sure the extension is **Enabled** (toggle should be on)
  - Click the puzzle icon and pin the extension

### "Could not load icon" error
- **Solution**: 
  - Make sure `2023_Facebook_icon.svg.png` exists in the folder
  - Reload the extension
